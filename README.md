# Facebook Clone Authentication System

A responsive Facebook-inspired authentication and profile dashboard application built using **HTML, CSS, and JavaScript**. This project allows users to create an account, log in securely, view their profile information, and manage their session through a simple dashboard interface.

## 🚀 Features

### User Registration

* Create a new account with personal information.
* Form validation for required fields.
* Success alert after account creation.
* User data stored locally in the browser.

### User Login

* Login using registered email and password.
* Credential verification before granting access.
* Error handling for invalid login attempts.

### User Dashboard

* Displays user profile information.
* Shows registered details such as:

  * Full Name
  * Email Address
  * Date of Birth
  * Gender
  * Join Date
* Dynamic avatar generated from the first letters of the user's name (e.g., **Taha Siddiq → TS**).

# Session Management

* User remains logged in during the active session.
* Logout functionality available in the navigation bar.
* Session data cleared after logout.
* Redirects users back to the registration/login page.

## 🔄 Application Flow

1. User opens the application.
2. Creates a new account using the registration form.
3. Account data is stored in browser storage.
4. Success message confirms account creation.
5. User navigates to the login page.
6. Enters email and password.
7. Authentication is performed.
8. Upon successful login:

   * User is redirected to the dashboard.
   * Profile information is displayed.
   * Avatar initials are generated automatically.
9. User can logout from the navbar.
10. Session is cleared and the user is redirected back to the authentication page.
